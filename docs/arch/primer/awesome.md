# Awesome Solution - 解决方案

::: info ℹ️

这里放置个人接触到的解决方案

:::

## IM / 即时聊天

- [OpenIM](https://www.openim.io/zh) - 开源、通用的即时通讯组件
- [WuKongIM](https://githubim.com/) - 高性能分布式即时通讯服务
- [bluexsx/box-im](https://github.com/bluexsx/box-im) - 盒子IM是一个仿微信的聊天工具

## 评论

- [Artalk](https://artalk.js.org/) - Arttalk 评论系统
- [giscus](https://giscus.app/zh-CN) **-** 利用 [GitHub Discussions](https://docs.github.com/zh/discussions) 实现的评论系统

## 手机

::: info 🌐

根据 UA 提取手机型号
- <a href="https://github.com/KHwang9883/MobileModels">KHwang9883/MobileModels</a> - 手机品牌型号汇总
- <a href="https://www.gsmarena.com/">GSMARENA</a> - 手机型号大全
- <a href="https://storage.googleapis.com/play_public/supported_devices.html">Google Play Supported Devices</a> - Google Play 支持的设备型号

:::

## 推荐

- [Gorse](https://github.com/gorse-io/gorse) - 开源的推荐引擎，会自动对数据进行训练，为每个用户生成推荐，方便接入各种在线服务
