# 使用 Spring Boot 开发

本节将更详细地介绍如何使用 Spring Boot，涵盖以下主题：

- **构建系统**：学习如何设置和管理项目的构建过程。
- **自动配置**：了解 Spring Boot 的自动配置功能，如何通过基于依赖项自动配置 Bean 和设置来简化设置。
- **运行应用程序**：发现启动和管理 Spring Boot 应用程序的多种方法。
- **最佳实践**：探索一些推荐的实践，以简化和改进 Spring Boot 开发。

虽然 Spring Boot 只是一个普通的库，但遵循一些指导原则可以大大简化开发体验。

**提示**：如果您刚开始使用 Spring Boot，建议在深入学习本节内容之前，先阅读 开发你的第一个 SpringBoot 应用 教程
