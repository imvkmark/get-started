# 国际化

Spring Boot 支持本地化消息，以便您的应用程序能够适应不同语言偏好的用户。默认情况下，Spring Boot 会在 `classpath` 的根目录下查找 `messages` 资源包。

::: info ℹ️
当配置的资源包的默认属性文件（默认是 `messages.properties`）可用时，自动配置生效。如果您的资源包仅包含特定语言的属性文件，您需要添加默认的文件。如果没有找到与任何已配置的基础名称匹配的属性文件，则不会自动配置 `MessageSource`。
:::

资源包的基名以及其他多个属性可以通过 spring.messages 命名空间进行配置，如下例所示：

```XML
spring.messages.basename=messages,config.i18n.messages
spring.messages.fallback-to-system-locale=false
```

::: tip 💡
`spring.messages.basename` 支持逗号分隔的多个位置列表，可以是包限定符，也可以是从类路径根目录解析的资源。
:::

有关更多支持的选项，请参阅 [`MessageSourceProperties`](https://docs.spring.io/spring-boot/api/java/org/springframework/boot/autoconfigure/context/MessageSourceProperties.html)
