# 5. 继承

## 5.1. 类, 超类和子类

继承是 `is-a` 的明显特征

### 5.1.1 定义子类

超类和子类的名称来源于计算机科学中的集合术语: 超集和子集

- 超类(super class), 基类(base class), 父类 (parent class)
- 子类(sub class), 派生类(derived class), 孩子类(child class)

### 5.1.2 覆盖方法

使用 `super.getSalary()` 调用父类方法, 使用 `super()` 调用父类的构造函数
