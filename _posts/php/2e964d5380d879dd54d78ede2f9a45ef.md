## 问题
> 你想使用缓存加速你的应用





## 解决方案
> 安装 Redis

```
$ sudo apt-get install -y redis-server
```
第一行安装 apc 包, 第二行重启 Apache.

## 讨论
现在, 你可以使用 redis 服务器 来配置你的缓存.

查看 `Setting up the Redis Cache Driver`  的说明.
